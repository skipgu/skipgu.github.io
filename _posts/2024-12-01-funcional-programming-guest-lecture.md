SKIP hosted an insightful functional programming guest lecture on 27th November 2024, in Alfons - Hus Patricia, with the assistance of senior lecturer and member of the Computer Science and Engineering (CSE) department - Alex Gerdes. 

![FP Guest Lecture](/assets/posts/guest-lecture-functional-programming-2024/fp-guest-lecture-1.png)

![FP Guest Lecture 2](/assets/posts/guest-lecture-functional-programming-2024/fp-guest-lecture-2.png)

Functional programming is a declarative programming paradigm that focuses on creating functions that avoid side-effects and is based on immutable datatypes. It is utilised in many safety-critical systems.

The lecture consisted of functional programming practices and patterns for modular and maintainable code for software engineering as well as a live-coding session to connect the attendees’ new theoretical foundations with practical applications. 

You can find further information about the live-coding session as well as the detailed lecture slides which were shown at the event [here](https://github.com/skipgu/fp4swe). 

Thank you for coming to the guest lecture!
